Installation
==============

At this time, RDG only supports Unix-like systems.

Pre-requisites
-----------------
- CMake
- gcc
- flex
- bison
- Python development headers (optional, required from python bindings)

Steps
---------------------

#) Download the source from the `RDG releases`_ on Gitub:

#) Extract the archive

#) Change to the extracted directory and run the following:

.. code-block:: bash

   mkdir build
   cd build
   cmake ..
   make
   make install

.. _RDG releases: https://github.com/ReverentEngineer/rdg/releases
