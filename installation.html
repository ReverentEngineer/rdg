Installation¶

At this time, RDG only supports Unix-like systems.

Pre-requisites¶

  • CMake

  • gcc

  • flex

  • bison

  • Python development headers (optional, required from python bindings)

Steps¶

  1. Download the source from the RDG releases on Gitub:

  2. Extract the archive

  3. Change to the extracted directory and run the following:

mkdir build
cd build
cmake ..
make
make install
